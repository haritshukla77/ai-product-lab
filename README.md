## 🚀 Projects

### 🤖 [AI News Agent](./AI-News-Agent/)

**Automated Daily AI News Digest System**

An intelligent news aggregation system that curates and delivers personalized AI/LLM news from trusted sources directly to your inbox every morning.

**Key Features:**
- 📰 Multi-source news aggregation (TechCrunch, VentureBeat, MIT Technology Review, ArXiv)
- 🎯 Intelligent keyword-based content filtering
- 📧 Beautiful HTML email formatting
- ⏰ Automated daily scheduling via macOS LaunchAgent
- 🔒 Secure Gmail SMTP integration
- 📊 Comprehensive logging and error handling

**Tech Stack:**
- Python 3.14
- RSS Feed Parsing (feedparser)
- Gmail SMTP
- APScheduler
- Jinja2 Templates
- BeautifulSoup4

**Highlights:**
- 7,000+ lines of production-ready code
- Modular architecture with separation of concerns
- Extensive documentation and setup guides
- Retry mechanisms and robust error handling
